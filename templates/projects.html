Zillow API Tool

A web application that connects to Zillow's API to get the information such as estimated value, longitute, zip code for a list of properties you upload. You can upload a csv file containing the street address, city and state, and it will return an output csv file with the additional information for the addresses. This is useful in helping institutional investors in the real estate space estimate the value of their own portfolios or the the portfolios they underwrite.
